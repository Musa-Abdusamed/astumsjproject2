

# 📝 To-Do List App

A simple and interactive To-Do List web application built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks with features like add, delete, mark as done, and persistent storage using `localStorage`.

## ✨ Features

- ➕ Add new tasks
- ✔️ Mark tasks as completed / undo tasks
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks
- 🔍 Prevent duplicate tasks
- 💾 Save tasks in browser (localStorage)
- 🎉 Celebration message when all tasks are completed
- 🎨 Background color theme switcher
- 📊 Live task counter (remaining & completed tasks)

## ⚙️ How It Works

1. User adds a task  
2. Task is stored in an array (`tasks`)  
3. UI is updated using `render()` function  
4. Data is saved in `localStorage`  
5. On reload, tasks are loaded again automatically
